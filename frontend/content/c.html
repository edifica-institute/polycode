C Standard Library — Functions • Use • Syntax • Examples

This reference covers common C standard headers. Each section shows description, key functions, syntax, and short example programs. Row-wise layout for easy reading/printing.

stdio.h — Standard I/O

Description: Input/output functions for console and file operations.

Syntax: printf, scanf, getchar, putchar, fopen, fclose, fread, fwrite

Example:

#include <stdio.h>
int main() {
    printf("Enter number: ");
    int n; scanf("%d", &n);
    printf("You entered %d\n", n);
    return 0;
}

stdlib.h — Utilities

Description: Memory allocation, conversions, random numbers, program control.

Syntax: malloc, calloc, free, rand, srand, exit, atoi, atof

Example:

#include <stdlib.h>
#include <stdio.h>
int main() {
    int *arr = malloc(5 * sizeof(int));
    for(int i=0;i<5;i++) arr[i] = rand()%10;
    free(arr);
    return 0;
}

string.h — String handling

Description: Functions for manipulating strings and memory blocks.

Syntax: strlen, strcpy, strcat, strcmp, memcpy, memset

Example:

#include <string.h>
#include <stdio.h>
int main() {
    char s1[20] = "Hello", s2[] = "World";
    strcat(s1, s2);
    printf("%s\n", s1); // HelloWorld
    return 0;
}

math.h — Math library

Description: Standard math functions.

Syntax: sqrt, pow, sin, cos, log, fabs

Example:

#include <math.h>
#include <stdio.h>
int main() {
    double r = sqrt(25.0) + pow(2,3);
    printf("%f\n", r); // 13.000000
    return 0;
}

time.h — Date & Time

Description: Get current time, measure intervals.

Syntax: time, difftime, clock, localtime, strftime

Example:

#include <time.h>
#include <stdio.h>
int main() {
    time_t now = time(NULL);
    printf("Now: %s", ctime(&now));
    return 0;
}

ctype.h — Character handling

Description: Character classification and conversion.

Syntax: isalpha, isdigit, isspace, toupper, tolower

Example:

#include <ctype.h>
#include <stdio.h>
int main() {
    char c = 'a';
    if(isalpha(c)) printf("%c is alphabet\n", c);
    printf("Upper: %c\n", toupper(c));
    return 0;
}
