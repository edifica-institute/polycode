C++ Standard Library — Functions • Use • Syntax • Examples

This reference covers common C++ standard headers. Each section shows description, key classes/functions, syntax, and short examples. Row-wise layout for easy reading/printing.

iostream — Standard I/O

Description: Console input/output streams.

Syntax: std::cout, std::cin, std::cerr

Example:

#include <iostream>
using namespace std;
int main() {
    int x; 
    cout << "Enter number: ";
    cin >> x;
    cout << "You entered " << x << endl;
}

fstream — File I/O

Description: File input/output using stream classes.

Syntax: std::ifstream, std::ofstream, std::fstream

Example:

#include <fstream>
#include <iostream>
using namespace std;
int main() {
    ofstream out("data.txt");
    out << "Hello File";
    out.close();

    ifstream in("data.txt");
    string s; in >> s;
    cout << s;
}

string — Strings

Description: The std::string class for text handling.

Syntax: length(), substr(), find(), replace(), concatenation with +

Example:

#include <string>
#include <iostream>
using namespace std;
int main() {
    string s = "Hello";
    s += " World";
    cout << s.substr(0,5); // Hello
}

vector — Dynamic arrays

Description: Resizable array container.

Syntax: push_back(), at(), size(), range-based for

Example:

#include <vector>
#include <iostream>
using namespace std;
int main() {
    vector<int> v = {1,2,3};
    v.push_back(4);
    for(int x : v) cout << x << " ";
}

map / set — Associative containers

Description: Ordered collections: map stores key-value, set stores unique values.

Syntax: insert(), find(), erase(), range iteration

Example:

#include <map>
#include <iostream>
using namespace std;
int main() {
    map<string,int> m;
    m["a"] = 1; m["b"] = 2;
    for(auto &p : m) cout << p.first << ":" << p.second << endl;
}

algorithm — Algorithms

Description: Generic algorithms for containers.

Syntax: sort(), find(), for_each(), count(), max_element()

Example:

#include <algorithm>
#include <vector>
#include <iostream>
using namespace std;
int main() {
    vector<int> v = {5,2,9,1};
    sort(v.begin(), v.end());
    for(int x : v) cout << x << " ";
}

utility — Pairs & Tuples

Description: Helper types for grouping values.

Syntax: std::pair, std::tuple, std::make_pair, std::tie

Example:

#include <utility>
#include <iostream>
using namespace std;
int main() {
    pair<int,string> p = {1, "one"};
    cout << p.first << "-" << p.second;
}

memory — Smart Pointers

Description: Automatic memory management.

Syntax: unique_ptr, shared_ptr, make_unique, make_shared

Example:

#include <memory>
#include <iostream>
using namespace std;
int main() {
    auto p = make_unique<int>(42);
    cout << *p;
}

chrono — Time

Description: Measure time, durations, clocks.

Syntax: steady_clock, system_clock, duration, now()

Example:

#include <chrono>
#include <iostream>
using namespace std;
using namespace std::chrono;
int main() {
    auto start = steady_clock::now();
    // ... work ...
    auto end = steady_clock::now();
    cout << duration_cast<milliseconds>(end-start).count();
}

thread / future / mutex — Concurrency

Description: Multithreading utilities.

Syntax: std::thread, std::mutex, std::lock_guard, std::async, std::future

Example:

#include <thread>
#include <iostream>
using namespace std;
void task(){ cout << "Hello from thread"; }
int main() {
    thread t(task);
    t.join();
}
