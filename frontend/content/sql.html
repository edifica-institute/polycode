Frontend Reference — HTML • CSS • JavaScript

This document lists common HTML elements/attributes, CSS selectors/properties, and JavaScript (core & DOM) with syntax and examples. Static, no scripts, no background colors.

HTML — Elements

ElementSyntaxExample
<div>, <span> <div>block</div>, <span>inline</span> 
<div class="card">Hello <span>there</span></div>
Semantic: header, nav, main, section, article, aside, footer <main><section><article>...</article></section></main> 
<header>Site</header>
<main><section><article>Post</article></section></main>
<footer>©</footer>
Headings & paragraphs <h1>..</h1> … <h6>, <p>text</p> 
<h1>Title</h1>
<p>A paragraph.</p>
<a> (anchor) <a href="path">text</a> 
<a href="#faq">Jump to FAQ</a>
<img> <img src="x.png" alt="desc" width=".." height=".."> 
<img src="logo.png" alt="Company logo" width="120" height="40">
Lists <ul><li>..</li></ul>, <ol>, <dl><dt><dd> 
<ol><li>Step 1</li><li>Step 2</li></ol>
Tables <table><thead><tbody></tbody></thead></table> 
<table>
  <thead><tr><th>Name</th><th>Score</th></tr></thead>
  <tbody><tr><td>Ana</td><td>92</td></tr></tbody>
</table>

CSS — Selectors

SelectorSyntaxExample
Basic & grouping p, .card, #hero, h1, h2 
.card { padding: 12px; }
Combinators .menu > li, h2 + p, h2 ~ p, .card p 
.menu > li { list-style: none; }
Attribute selectors input[type="email"], a[href^="#"], a[href$=".pdf"] 
a[href^="tel:"] { text-decoration: none; }
Pseudo-classes/elements :hover, :focus-visible, :not(), :nth-child(), ::before, ::after 
.tag::before { content: "• "; }

CSS — Box Model & Layout

TopicSyntaxExample
Padding/Margin/Border/Box-sizing padding: t r b l;, margin:, border:, box-sizing:border-box; 
.panel { padding: 12px 16px; box-sizing: border-box; }
Width/Height & Min/Max width:, max-width:, min-height: 
img { max-width: 100%; height: auto; }
Display & Position display:block|inline|flex|grid, position:relative|absolute|sticky 
.sticky { position: sticky; top: 0; }
Flexbox display:flex, gap, justify-content, align-items, flex

Grid display:grid, grid-template-columns, gap 
.grid { display:grid; grid-template-columns: repeat(3, 1fr); gap: 14px; }

CSS — Typography & Effects

TopicSyntaxExample
Fonts & Text font-family, font-size, font-weight, line-height, text-align 
h1 { font-size: 24px; line-height: 1.3; }
Color & Background color, background-color, background-image: linear-gradient(...) 
.btn { color: #fff; }
Transforms/Transitions transform, transition, @keyframes 
.btn { transition: transform .15s ease; }
.btn:hover { transform: translateY(-1px); }
Media/Feature/Container queries @media, @supports, @container 
@media (max-width:640px){ .grid{ grid-template-columns:1fr; } }

JavaScript — Core

TopicSyntaxExample
let / const / var let x = 1;, const PI = 3.14;, var old; 
const name = "Dip"; let count = 0;
Functions & Arrows function f(a){}, const f = (a) => {} 
const add = (a,b) => a + b;
Arrays — map/filter/reduce arr.map, arr.filter, arr.reduce, includes, find 
[1,2,3].map(x => x*2) // [2,4,6]
Strings — replace/pad/split replace, replaceAll, padStart, split, startsWith 
"Hello".padStart(7,"_") // "__Hello"
Objects & JSON Object.keys/values/entries, JSON.parse/stringify 
const o = {a:1}; const p = {...o, b:2};
Promises / async–await new Promise(), async function, await 
async function load(){ return 42; }
