SQL Functions and Keywords Reference (MySQL/SQLite)

This document lists important SQL functions and reserved keywords with syntax and examples, organized by category.

Aggregate Functions

FunctionSyntaxExample
COUNTCOUNT(*) or COUNT(expr)SELECT COUNT(*) FROM employees;
SUMSUM(expr)SELECT SUM(salary) FROM employees;
AVGAVG(expr)SELECT AVG(age) FROM students;
MINMIN(expr)SELECT MIN(price) FROM products;
MAXMAX(expr)SELECT MAX(score) FROM results;
GROUP_CONCATGROUP_CONCAT(expr SEPARATOR ',')SELECT GROUP_CONCAT(name) FROM employees;

FunctionSyntaxExample

Window Functions

FunctionSyntaxExample

Note: SQL syntax and functions vary slightly between database systems (MySQL, PostgreSQL, SQL Server, Oracle, SQLite). This reference is based on common MySQL 8.0 style usage.
