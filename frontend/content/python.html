Python Core Modules — Modules • Functions • Use • Syntax • Examples

This reference highlights widely used Python standard library modules and built-ins. Each row shows purpose, syntax, and a short example. Static, no scripts, print-friendly.

Built-ins — Core Functions & Types

Function / TypeUseSyntaxExample
print Output to console print(*objects, sep=' ', end='\\n') 
print("Hello", "World", sep=", ")  # Hello, World
len Length of object len(obj) 
len([1,2,3])  # 3
range Sequence of numbers range(stop), range(start, stop, step) 
for i in range(1,5): print(i)
enumerate Index + value pairs enumerate(iterable, start=0) 
for i,v in enumerate(['a','b']): print(i,v)
map, filter Apply function / filter items map(func, iterable) • filter(func, iterable) 
list(map(str.upper, ["a","b"]))
