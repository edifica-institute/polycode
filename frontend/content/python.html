Python Core Reference — Built-ins • Data Types • Modules • Examples

Reference of Python built-ins, data types, OOP, and key libraries like NumPy, Pandas, Matplotlib, Statistics. Row-wise cards for readability.

Strings

str Immutable text sequence s.upper() • s.lower() • s.split() • s.replace(a,b) 
"Hello".replace("H","Y")  # 'Yello'

Lists

list Mutable ordered collection append • extend • remove • sort 
nums = [1,2,3]; nums.append(4)

Tuples

tuple Immutable ordered collection (a,b,c) 
t = (1,2,3); print(t[0])

Dictionaries

dict Key-value mapping d[key] = value • d.get() • d.items() 
d = {"a":1}; print(d["a"])

Sets

set Unique unordered elements s.add(x) • s.union() • s.intersection() 
{1,2,3} | {3,4}  # {1,2,3,4}

OOP — Classes & Objects

class Blueprint for objects class Name: def __init__(...): ... 
class Dog: 
  def __init__(self,n): self.name=n
d = Dog("Tommy")
Inheritance Reuse & extend classes class Sub(Base): ... 
class Puppy(Dog): pass

LEGB Rule — Scopes

LEGB Name lookup order Local → Enclosing → Global → Built-in 
x="global"
def outer(): x="enclosing"
  def inner(): x="local"; print(x)
  inner()
outer()

NumPy — Arrays

ndarray Efficient n-D arrays np.array() • reshape • dtype 
import numpy as np
a = np.array([1,2,3])
2D Array Matrix-like data np.array([[...],[...]]) 
m = np.array([[1,2],[3,4]])

Pandas — Data Analysis

Series 1D labeled array pd.Series(data) 
import pandas as pd
s = pd.Series([1,2,3])
DataFrame 2D labeled table pd.DataFrame(dict) 
df = pd.DataFrame({"a":[1,2],"b":[3,4]})

Matplotlib — Pyplot

plot Basic line plotting plt.plot(x,y) • plt.show() 
import matplotlib.pyplot as plt
plt.plot([1,2,3],[4,5,6]); plt.show()
bar / scatter Bar/scatter plots plt.bar() • plt.scatter() 
plt.bar([1,2,3],[3,4,5])

statistics — Data Stats

mean / median / mode Basic statistics statistics.mean(data) 
import statistics as st
print(st.mean([1,2,3]))
stdev / variance Spread measures statistics.stdev(data) 
st.stdev([10,20,30])
