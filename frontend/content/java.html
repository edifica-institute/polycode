Java Core Packages — Packages • Functions • Use • Syntax • Examples

This reference focuses on widely used Java SE packages. For each package, key classes & functions (methods) are summarized with syntax and short examples. Static, no scripts, print-friendly.

Overview — Common Java SE Packages

Package / ModuleUseNotes
java.lang (java.base) Core language classes: Object, String, Math, System, Thread, exceptions Auto-imported (no import needed)
java.util (java.base) Collections, Optional, Random, Scanner, utility classes Lists, Maps, Sets, algorithms
java.time (java.base) Modern date/time API: LocalDate, Instant, Duration, Formatter Immutable, timezone-safe types
java.io (java.base) Streams, Readers/Writers for classic I/O Often paired with java.nio.file
java.nio.file (java.base) NIO.2 filesystem: Paths, Files Convenient static methods (read/write/copy/walk)
java.net / java.net.http Networking (URL, URI, Inet), HTTP client (JDK 11+) HttpClient, HttpRequest, HttpResponse
java.util.concurrent Threads, executors, locks, concurrent collections Futures, CompletableFuture, pools
java.util.stream Stream API for functional operations on collections map/filter/reduce, collectors
java.math BigInteger, BigDecimal for precise math Immutable arbitrary precision
java.sql JDBC database access Driver-specific URL and dependencies
java.util.regex Regular expressions (Pattern, Matcher) Java-style regex syntax
