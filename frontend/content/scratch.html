Scratch — Quick Reference

This page is static and print‑friendly. Use it as a left‑pane guide while you create projects in the center editor.

Getting Started

  1. Use the box above to paste a Scratch/TurboWarp Project ID or URL, then click Load.
  2. Or start from a blank editor. Add sprites, backdrops, and blocks.
  3. Click the green flag to run. Click the stop button to halt.
  4. Save: File → Save to your computer (downloads .sb3).
  5. Open: File → Load from your computer and pick an .sb3.

Projects never leave your browser unless you choose to upload/download inside the editor.

Editor Anatomy

  • Stage: Preview/play area at right.
  • Sprites: Characters/objects you control.
  • Blocks: Drag from the palette to the scripting area.
  • Backdrops: Background images for the stage.

Handy Shortcuts

  • Ctrl/Cmd + Z / Y — Undo / Redo
  • Ctrl/Cmd + S — Save (via File menu action)
  • Ctrl/Cmd + C / V — Copy / Paste blocks
  • Delete — Remove selected block or sprite

Importing Projects

  • ID/URL: Paste the Scratch or TurboWarp project URL/ID in the box above and press Load.
  • File: Use File → Load from your computer to open .sb3.

Tip: You can remix public Scratch projects by loading their ID.

Export / Share

  • Save to computer: Downloads .sb3 (editable).
  • Player link: Use TurboWarp embed (read‑only play view) if you just want to share a playable version.
  • Performance: In TurboWarp, you can enable high FPS/turbo mode for smoother animations.

Troubleshooting

  • Editor not visible? Make sure your iframe URL points to /tw/editor.html?embed=1 on your own site.
  • 404 on /tw/editor.html: Ensure the tw/ folder exists on your host and _redirects contains /tw/* /tw/index.html 200.
  • Can’t save? Browser may block downloads in iframes; try opening the editor in a new tab (right‑click → Open frame in new tab) just for saving.
