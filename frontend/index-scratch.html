Scratch — Tips & Shortcuts

Quick Start

  1. Use the menu inside the editor (top-left) to Load from your computer or Save to your computer (SB3 files).
  2. Paste a Scratch/TurboWarp project URL or numeric ID above and click Load.
  3. Click the green flag in the editor to run your project.

Keyboard

  • Ctrl/Cmd + S — Save (inside editor menu)
  • Ctrl/Cmd + Z / Y — Undo/Redo

Notes

  • Editor is powered by TurboWarp (modern Scratch 3 runtime) for speed & compatibility.
  • Files never leave your browser unless you choose to upload/download in the editor.
